# TodoList Application

## Description
This TodoList application is a simple yet powerful tool to help users manage their daily tasks. It provides a clean interface for users to add, delete, and view their tasks with ease. Built with HTML5, CSS3, and JavaScript, this app is designed for those who want to stay organized and productive.

## Features
- **Add Tasks:** Users can add new tasks to their list.
- **Delete Tasks:** Users can remove tasks once they are completed.
- **View Tasks:** All tasks are displayed in an easy-to-read list format.
